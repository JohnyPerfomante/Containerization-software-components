# Flask + PostgreSQL Dockerized App

Цей проєкт слугує для демонстрації, як розгорнути простий веб-додаток на **Flask** з підключенням до бази даних **PostgreSQL**, використовуючи **Docker Compose**.

---

## Технології

- **Python 3.11 + Flask** — веб-фреймворк для створення REST API
- **PostgreSQL 15** — реляційна база даних
- **Docker Compose** — для оркестрації контейнерів
